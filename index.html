Projects

Wake up call for aquatic enthusiasts using YOLO V8

The model was developed using YOLO V8. The model was trained with the dataset which was prepared by own with various aquatic classes. This model will detect the alert the aquatic enthusiasts (scuba drivers, swimmers) when harmful animal nears them.

Cleanliness Automation

The model will alert the coroperation authorities when it detect the overflow of the garbage. The Solution was built in both hardware and software. In hardware part the sensor is placed inside the trash can, it measures the distance from it to garbage level and in software side it was trained using YOLOV5.
